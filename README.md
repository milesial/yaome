# yaome - Yet another online Markdown editor

> Work in progress

- **pandoc** full power
- **real-time** preview
- **multiple files**
- **customizable** configuration
- **citations** support
- **clean** design


Built with [Vue](https://vuejs.org/) and [Node](https://nodejs.org/en/about/)
